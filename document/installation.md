This document details the libraries and environment required to run the Universal Finance Tracker.

### 1. Library Dependencies
We installed the following Python libraries:
- **streamlit**: The web framework for the dashboard.
- **pandas**: For data manipulation and cleanup.
- **pdfplumber**: To extract text and tables from PDF statements.
- **plotly**: For interactive charts and visualizations.
- **fpdf2**: To generate the executive PDF reports.
- **kaleido**: Required to convert Plotly charts into static images for the PDF.
- **openpyxl**: Enables pandas to read Excel (.xlsx) files.

### 2. Environment Setup
- **Python 3.10+**: Recommended version.
- **Virtual Environment (venv)**: Used to keep dependencies isolated from the system.
